# Furina-skillhub

Personal collection of Claude Code skills.

| Skill | What it does |
|---|---|
| [`xiaohongshu-publish`](skills/xiaohongshu-publish) | Publish a note (title/body/cover image/tags) to Xiaohongshu via browser automation. |
| [`xiaohongshu-paper-explainer`](skills/xiaohongshu-paper-explainer) | Turn an arXiv paper into a Xiaohongshu paper-explainer card deck, using the paper's own figures. Builds on `xiaohongshu-publish`. |
| [`hf-daily-papers`](skills/hf-daily-papers) | Read and summarize HuggingFace daily papers. |
| [`gs-search`](skills/gs-search) | Search Google Scholar by keywords. |
| [`gs-advanced-search`](skills/gs-advanced-search) | Google Scholar search with filters (author, journal, date range, etc). |
| [`gs-cited-by`](skills/gs-cited-by) | Find papers that cite a given Google Scholar paper. |
| [`gs-export`](skills/gs-export) | Export a Google Scholar paper to Zotero via BibTeX. |
| [`gs-fulltext`](skills/gs-fulltext) | Get full-text access links (PDF/DOI/publisher) for a paper. |
| [`gs-navigate-pages`](skills/gs-navigate-pages) | Navigate Google Scholar search result pages. |

## Install

Copy a skill folder into `~/.claude/skills/`.
